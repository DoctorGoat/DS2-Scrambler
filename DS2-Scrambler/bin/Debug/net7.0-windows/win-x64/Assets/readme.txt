# GenerateType
#--------------------------------------
# Generate - Used to generate new values
Generate folder contains the list of fields for a param that should be generated with the specified minimum and maximum values.

For example:
base_price;10;10000;

The field "base_price" will be given a random value between 10 and 10000.

# Shuffle - Used to shuffle existing values
Shuffle folder contains the list of fields for a param that should be shuffled (whilst using GenerateType).

For example:
speffect_id

The field speffect_id values will be shuffled. 

#--------------------------------------
# ShuffleType
#--------------------------------------
ShuffleType folder contains the list of fields for a param that should be shuffled.

For example:
base_price

The field base_price values will be shuffled.
